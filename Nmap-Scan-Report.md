# Nmap Network Scan Report

## Objective
The goal of this project is to perform network scanning using Nmap and identify active services.

## Tools Used
- Kali Linux
- Nmap

## Methodology
A network scan was performed to identify active hosts and open ports. Both TCP and UDP scans were used.

## Findings
- Port 53/UDP is open
- Service: DNS
- Detected as Cloudflare Public DNS

## Analysis
The presence of an open DNS port indicates that the system is using an external DNS service. Some ports appeared filtered, suggesting firewall protection.

## Conclusion
This project demonstrates basic network scanning skills and understanding of port states and services.
